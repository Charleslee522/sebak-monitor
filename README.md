# sebak-monitor

------------------------

### Installation

* python: 3.6 or higher

### Deployment

```
$ python sebak-monitor.py
```

### Q & A

1. When you want to change checking nodes interval?
   Set `interval_sec` variable in sebak-monitor.py
1. When you want to alarm with email?
   Modify `alarm with email and slack` part in sebak-monitor.py

These configuration is going to change to `INI`
